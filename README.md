# Secure-Exam-Proctor-Grader

An AI-powered online examination platform designed to conduct secure assessments with automated proctoring and grading capabilities. The system helps detect suspicious activities during online exams while providing efficient exam management and evaluation.

## Features

* Secure online examination platform
* AI-based proctoring system
* Real-time suspicious activity detection
* Automated grading system
* Student and admin authentication
* Exam management dashboard
* Frontend and backend integration

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js / Express.js
* Python

### AI & Computer Vision

## Key Functionalities

### AI-Based Proctoring

The system monitors candidates during online exams and detects:

* Multiple face detection
* Mobile phone usage
* Suspicious movements
* Absence from screen

### Automated Grading

Supports automatic evaluation and result processing for online assessments.

### Authentication System

Role-based authentication for:

* Students
* Administrators
